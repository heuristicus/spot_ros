Robot Configuration¶

To configure the robot, you should consult Boston Dynamics’ support pages.

Robot administration is performed through the admin console.

Networking¶

See the network setup pages for details on network configuration.

Using WiFi for ROS communication is not recommended, except for monitoring purposes.

The driver can output a lot of data. It is recommended that you connect the Spot to the ROS PC via a payload port. See the payload developer guide for details.
